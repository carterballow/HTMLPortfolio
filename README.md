## First Iteration of Portfolio Webpage


## About

A single-page portfolio showcasing my education, skills, projects, and extracurriculars. Built as a first iteration to establish an online presence and share my work with recruiters, collaborators, and friends.

## Live Site

[carterballow.github.io](https://carterballow.github.io/HTMLPortfolio/)

## Tech Stack

- **HTML5** — markup and page structure
- **CSS3** — custom properties, grid/flexbox layouts, responsive design
- **JavaScript** carousel logic, scroll, UI interactions
- **Swiper.js** — slider/carousel support
- **Unicons (Iconscout)** — icon library

## Sections

1. **Home** — intro, headshot, social links
2. **About** — bio, GPA, certifications, languages, and resume download
3. **Skills** — expandable categories covering technical, development, policy, and soft skills
4. **Education** — timeline from middle school through UCLA
5. **Endeavors** — carousel of current projects and activities
6. **Contact** — reach out via phone, email, or the contact form!
